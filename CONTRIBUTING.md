# Contributing

This repo is an early public prototype. Contributions should keep the project
plain-language, evidence-based, and useful for African country teams.

Useful contributions include:

- adding a DPG with an official GitHub link
- improving a use-case reference stack
- adding a country readiness note with sources
- improving safeguards guidance
- adding a sandbox API example
- making the interface clearer and easier to use

Do not add private meeting notes, personal contact details, unverified country
claims, or internal documents.
