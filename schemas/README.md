# Schemas

JSON Schemas for the AA4DPI registry.

- `open-dpg.schema.json`: one open DPG or open-source tool entry
- `cohort-country.schema.json`: one Cohort 1 country registry entry
- `stack-kit.schema.json`: one generated country/use-case starter kit

These schemas are intentionally small. They create enough structure for GitHub
review without pretending the AA4DPI operating model is finished.
